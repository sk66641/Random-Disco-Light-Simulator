About

The Random Disco-Light Simulator
It is a fun and interactive project that brings the excitement of a disco dance floor to your screen. Whether you're a developer, a party enthusiast, or just someone who loves vibrant colors, this simulator is designed to brighten up your day!
Inputs
1. Number of Colors : Specify how many random colors you want to be shown.
2. Time Interval : Set the interval (in milliseconds) at which the colors should change.
3. View Type : Choose from conic, linear, or radial views to customize the visual effect.
4. Countdown Timer : Set a timer (in seconds) for how long the simulation should run.
5. Sound Effect : Optionally, add a sound effect to enhance the disco experience.
Output
Based on the inputs you provide, the simulator will display a dynamic and colorful light show in the selected view, changing colors at the specified intervals. The simulation will continue until the countdown timer expires.
Examples
Try these input combinations and see the magic unfold!
1. 1000 colors, 1 millisecond interval, conic view, 60 seconds timer.
2. 1000 colors, 1 millisecond interval, radial view, 60 seconds timer.
3. 1000 colors, 1 millisecond interval, linear view, 60 seconds timer.
